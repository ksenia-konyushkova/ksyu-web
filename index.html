Current research

Currently I am working on active learning for different segmentation tasks. While the goal of machine learning in general is to find the best predictive model given some data, the goal of active learning is to select which data should be annotated in order to learn the model as quickly as possible.

In practice, this means that instead of asking humans to segment whole images, we adaptively select which parts should be labelled in priority. It helps to reach the same quality of segmentation as standard machine learning methods in a shorter time.
